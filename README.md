# Projeto Java Web Services Com Spring Boot e JPA/Hibernate

## Sobre o projeto

API REST de um Delivery.

## As seguintes tecnologias foram utilizadas neste projeto:

* Java EE 17
* Spring Boot Framework (version 3.0.0)
* SQL
* JPA
* Hibernate
* JSON
* H2 Database
* PostgreSQL
* Maven
* REST
* Docker
* Git
* GitHub
* Postman
